# Website-Absensi-QRCode
Aplikasi absensi dengan QR Code adalah sistem yang memungkinkan pengguna untuk melakukan pencatatan kehadiran dengan memindai kode QR menggunakan perangkat seluler atau pemindai QR Code. Kode QR ini mengandung informasi mengidentifikasi pengguna dan waktu kehadiran mereka. Sistem ini berbasis web yang digunakan dengan localhost.
